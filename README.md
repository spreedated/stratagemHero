Stratagem Hero - from Helldivers 2
==================================

Your favorite game from Helldivers 2 is now available as a standalone game!

---

**Features**

- WASD & Arrow Key Navigation
- Local Highscores
- No Gameengine, just plain WPF
- Easy to use, hard to master
